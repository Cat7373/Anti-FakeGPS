# Anti-FakeGPS

模拟定位检测工具（只适用于移动端，因为电脑一般没有 GPS 硬件）



## 检测原理

1. 市面上大部分模拟定位器，都存在一个特征，即会将 GPS 锁定在一个特定位置上不再变动
2. 本包基于此特点，制作了自动化检测工具，并提供了适用于巡逻打卡拍照场景的工具方法
3. 更多细节，请参阅[博客文章](https://blog.cat73.org#TODO)了解更多
